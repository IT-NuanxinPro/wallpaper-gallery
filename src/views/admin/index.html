API 文档
GET /api/wallpapers?orientation=horizontal&style=自然_风景&color=偏蓝&page=1&limit=30
GET /api/random?type=url|img&orientation=&style=&color=&thumb=true&w=1920
GET /api/file/<key>?w=800&q=80
orientation: horizontal | vertical | square
style: 动漫_二次元 | 魅力_迷人 | 自然_风景 | 自制_艺术 | 游戏_玩具 | 安逸_自由 | 科幻_星云 | 程序_代码
color: 偏蓝 | 灰_白 | 暗色 | 紫_粉 | 偏黄 | 偏绿 | 偏红 | 其他颜色

Bot 配置

配置 Telegram Bot Token 与频道映射（存储在环境变量中，此处仅查看状态）

环境变量说明
在 CF Pages → 设置 → 环境变量 中配置以下变量：
WG_BOT_TOKENS = JSON格式，chatId 映射到 botToken
{"−1003814998799":"YOUR_BOT_TOKEN_1","−1003777271568":"YOUR_BOT_TOKEN_2","−1003718954682":"YOUR_BOT_TOKEN_3"}
ADMIN_USER = 管理员用户名
ADMIN_PASS = 管理员密码
ADMIN_SECRET = JWT 签名密钥（32位随机字符串）
频道状态检测
